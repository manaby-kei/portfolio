【休憩リマインダー PWA】ブラウザの制限を克服し、無料で運用可能な実用的なWebアプリケーション


・アプリの概要と技術的強み

本制作物は、作業集中力を高めるための自動休憩リマインダーです。Web技術だけで作った PWA なので、スマホのホーム画面へのインストールやオフライン動作も可能です。

技術的な優位性 (Technical Highlights)
スロットリング対策と正確な時間計測: 複数タブを開いている場合、アプリを非アクティブなタブで実行するとタイマーが遅延する問題に対し、setInterval に依存せず、開始時の絶対時刻を基準に残り時間を計算するロジックを採用。これにより、タブを切り替えても時間が正確に計測され、通知がほぼ遅延しない堅牢性を実現しました。

Web Push 通知によるユーザー体験: Web Notification APIとService Workerを利用し、ブラウザが最小化または非アクティブな状態でも、指定時刻に確実なデスクトップ通知（PC通知）を配信、実用的なリマインダーとしてしっかり機能します。

高度なUIとユーザビリティ: タイマー実行中も終了時刻を基準に即座に残り時間を調整できるプラス/マイナス/リセットボタンを実装。煩雑な入力作業を排除し、高い操作性を実現しました。

無料枠内での運用設計: 当初はスマホ連携も目指しましたが、Firebaseの無料プランでは実現が難しかったため、断念しました。Firebase Hosting の無料枠（Spark プラン）のみで PWA を運用できるアーキテクチャを採用。運用コストを意識し、この設計を採用しました。

・使用技術スタック

フロントエンド: HTML5 / CSS3 / JavaScript (ES6 Modules)

機能: Web Workers (Service Worker), Web Notification API, LocalStorage API

デプロイ/インフラ: Firebase Hosting (無料枠運用)
